What is HTML refer to?

- HTML refer to Hyper Text Markup Language which is a markup language for creating a webpage. Webpages are usually viewed in a web browser HTML can include writing, links, pictures, and even sound and video. HTML is used to mark and describe each of these kinds of content so the web browser can display them correctly

- HTML uses tags which usually come in pairs the opening tag (tell us more about the content of that element) refer to the start of a piece of content the closing tag refer to the end.
